# 🚀 SafariNAO Browser

SafariNAO is a modern, lightweight, and open-source web browser built using **Electron**, HTML5, and CSS3. Born and developed in **Italy** 🇮🇹, this project aims to provide a fluid browsing experience by stripping away the bloat and focusing on pure performance.

![SafariNAO Dashboard](https://drive.google.com/uc?export=download&id=1SLzjf_l-zFCDqix60Sz3V6alWrXgIFGB)

## 🇮🇹 The Italian Touch
SafariNAO is more than just a browser; it's an Italian open-source initiative. We focus on clean design (inspired by Mediterranean minimalism) and robust engineering to ensure a seamless experience for users worldwide.

## ✨ Key Features
* **Smart Dashboard**: Instant access to Google, YouTube, Prysma AI, and more with a single click.
* **Advanced Tab Management**: Dynamic tab system designed to prevent overlap, featuring quick-close and process isolation.
* **YouTube Rendering Fix**: Optimized frame handling to prevent partial loading or graphical glitches on media-heavy sites.
* **Mac-Style Custom UI**: Elegant window controls and a native Dark Mode for a professional look.
* **Optimized Drag-and-Drop**: Specifically engineered "safe zones" to ensure window dragging never interferes with tab navigation.
